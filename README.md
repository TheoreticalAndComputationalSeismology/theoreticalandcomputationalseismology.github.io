## Build 

```bash
jupyter-book build tcs/ --builder pdfhtml
cp tcs/_build/pdf/book.pdf tcs/book.pdf
jupyter-book build tcs/ --builder pdfhtml
```

Technically you can build the html without the pdf, but the pdf is nice to 
have.
